mac 下安装 nautilus

琰圭

对于文件管理器来说，我还是nautilus的忠实爱好者，至于为什么喜欢它，那是跟其他的相比，觉得它的可配置性更不错，更方便我这样有癖好的人。

mac 下默认的Finder 我是真的用不惯，不能方便的显示隐藏文件，不能自定义书签快捷定位(?) , 所以国庆一空，就忍不住黑科技一下。

郑传

mac下安装软件可以通过几种方式，app store, brew, 源码安装。 但nautilus 不支持这些安装方式。搜集得到一个有用的链接

  • Is it possible to run Nautilus on OS X?

通过MacPorts 安装 nautilus。

这里简述下步凑：

  • 安装MacPorts
    安装中文指南macports

    安装完后，记得更新 macports ，然后导出变量

    sudo port -v selfupdate

export PATH=$PATH:/opt/local/bin

  • 安装nautilus 

    sudo port install nautilus

问题汇总

nautilus 不能打开

成功安装nautilus后，但是却不能打开，打开错误如下:

(nautilus:48868): Gtk-WARNING **: cannot open display:

遇到这个问题，需要首先安装 xorg-server。

sudo port install xorg-server
