安装步凑

  1. 首先获取git上的源代码，因整个项目太大，暂时只获取8.0 分支上的代码

    git clone -b 8.0 git@github.com:odoo/odoo.git odoo_8.0

  2. 安装依赖

    sudo pip install -r requirements.txt

  3. 安装数据库， postgresql ,创建账户

    sudo su - postgres -c “createuser -s $USER”

  4. 安装样式

    sudo npm install -g less less-plugin-clean-css

  5. 启动项目，并在web界面创建数据库

    ./odoo.py –addons-path=addons

ps: odoo 基于python2.7 ,数据库采用postgresql

小帮助

  1. 打开开发者模式（调试模式）

    在左上角关于odoo窗体中，点击激活开发者模式

其他

除了上述odoo本身的内容外，还需要了解如下知识

  • xmlrpc
  • Python的XMLRPC简介及小技巧

注意

  • field 的某些属性存在顺序关系（即位置参数，这种情况一般出现在关系字段上）
  • 对象字段， 关联对象一定要有name 字段，odoo 默认以name显示对象名称，如果没有name字段，则显示对象本身
  • compute 与 function 的区别
